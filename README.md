# Проект: Сортировщик мусора на Raspberry Pi 4 (VLM + OpenCV)

Этот проект разработан для хакатона. Он использует камеру для обнаружения движения, нейросеть `moondream2` для классификации мусора и воспроизводит анимацию-инструкцию для пользователя.

## Структура проекта
- `main.py` — основной код (OpenCV + Logic).
- `setup_pi.sh` — скрипт автоматической настройки Raspberry Pi.
- `requirements.txt` — зависимости Python.
- `animations/` — папка для ваших видео-инструкций (.mp4).
- `llama.cpp/` — движок нейросети (будет собран скриптом).

## Как запустить на Raspberry Pi
1. Перенесите все файлы на Raspberry Pi.
2. Дайте права на выполнение скрипту настройки:
   ```bash
   chmod +x setup_pi.sh
   ```
3. Запустите настройку (это установит OpenCV, соберет llama.cpp и скачает модели):
   ```bash
   ./setup_pi.sh
   ```
4. Положите свои видео в папку `animations/`:
   - `plastic.mp4`
   - `organic.mp4`
   - `battery.mp4`
   - `error.mp4`
5. Запустите систему:
   ```bash
   python3 main.py
   ```

## Как работает логика
1. **Камера** следит за изменениями пикселей.
2. Когда в кадре появляется объект и замирает на 3 секунды, делается **фото**.
3. Фото отправляется в **moondream2**, которая отвечает одним словом (category).
4. Запускается **ffplay**, который показывает видео-инструкцию на весь экран.
5. После паузы система возвращается в режим ожидания.

## Оптимизация для 2GB RAM
- Чтобы избежать переполнения памяти, нейросеть вызывается как отдельный процесс (`llava-cli`).
- Используется квантованная модель `Q4_K` для быстродействия.
